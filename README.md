# AlphaByte Site

AlphaByte AIの企業サイトです。最先端のAI技術を活用したソフトウェア開発サービスを提供しています。

## 技術スタック

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Sparkles Effect
- Interactive Globe
- Gooey Text Morphing

## 機能

- モダンなUI/UXデザイン
- インタラクティブなアニメーション
- レスポンシブデザイン
- パフォーマンス最適化
- SEOフレンドリー

## 開発

```bash
# 依存関係のインストール
yarn install

# 開発サーバーの起動
yarn dev

# ビルド
yarn build
```
